# Changelog

All notable changes to `filament-smart-export` will be documented in this file.

## 1.0.0 - 2025-12-11

### Added
- Initial release
- Auto-discovery of models, columns, and relationships
- Grouped dropdowns by model
- Smart emoji assignment
- Live preview functionality
- Support for multiple relationships (HasMany, BelongsToMany)
- Export to XLSX and CSV formats
- Multilingual support (English and Spanish)
- Date range filters
- Zero configuration required
- Direct stream download without disk storage (no temp files)
